## Flask envronment settings

Powershell

```powershell
> $env:FLASK_APP = "hello"
> $env:FLASK_DEBUG = 1
> flask run
 * Running on http://127.0.0.1:5000/
```

BASH

```bash
$ export FLASK_RUN_PORT=8000
$ flask run
 * Running on http://127.0.0.1:8000/
```
